# koishi-plugin-febfrost-arknights-info

明日方舟干员信息推送插件。首版聚焦干员生日：每日凌晨向白名单群聊推送今日生日干员，周一推送本周每天生日干员，也支持命令手动查询。

- `今日生日干员` / `今日干员`：发送横向头像组合图，图片后接一段文本展示“今天生日的干员有...”并包含外文代号。
- `本周生日干员` / `本周干员`：发送本周生日干员图片，同日横向排列，不同日期纵向排列，今天的日期标题会高亮。
- `刷新干员生日缓存`：刷新今日和本周生日缓存。

数据来源：[ark-info](https://www.npmjs.com/package/ark-info)。

默认每日生日推送时间为 00:00。
